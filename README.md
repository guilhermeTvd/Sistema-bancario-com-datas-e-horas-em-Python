# 🏦 Sistema Bancário em Python

Um sistema bancário completo desenvolvido via terminal, focado na aplicação de conceitos avançados de Orientação a Objetos (POO).

## 🚀 Funcionalidades

* **Gestão de Clientes:** Criação de usuários (Pessoa Física) com proteção de dados sensíveis (Encapsulamento).
* **Gestão de Contas:** Vinculação de múltiplas contas correntes a um único cliente.
* **Operações Financeiras:** Saque, Depósito e visualização de Extrato detalhado.
* **Histórico com Timestamp:** Registro exato de data e hora em todas as transações utilizando a biblioteca `datetime`.
* **Regras de Negócio:** Limite de segurança de 10 transações diárias por conta, implementado de forma elegante utilizando **Decoradores** no Python.

## 🛠️ Tecnologias e Conceitos Aplicados

* Python 3
* Orientação a Objetos (Classes, Herança, Encapsulamento, Composição)
* Decoradores (`@property` e customizados)
* Git e GitHub

## 💡 Como executar no seu computador

1. Faça o clone deste repositório:
`git clone https://github.com/guilhermeTvd/Sistema-bancario-com-datas-e-horas-em-Python.git`

2. Execute o arquivo principal no seu terminal:
`python Sistema_bancario_com_datas_e_horas.py`
